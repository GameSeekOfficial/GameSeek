# ⚙️ How GameSeek is Built.

> *One developer. Two platforms. Every single day.*

-----

## 🌐 Website

The GameSeek website is built with a classic but solid stack — focused on performance and simplicity.

|Layer         |Technology           |
|--------------|---------------------|
|🎨 **Frontend**|HTML, CSS, JavaScript|
|🔧 **Backend** |Python               |

Clean, lightweight, and built to load fast.

-----

## 📱 App

The GameSeek app is built with a modern, component-driven stack designed for a smooth user experience.

|Layer         |Technology           |
|--------------|---------------------|
|⚛️ **Frontend**|React + TypeScript   |
|🎨 **Styling** |HTML, CSS, JavaScript|
|🔧 **Backend** |Python               |

Type-safe, scalable, and built to grow.

|Area              |Details                                  |
